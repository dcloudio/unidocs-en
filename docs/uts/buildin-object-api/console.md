## console

### debug
在控制台打印 debug 日志
```ts
console.debug(msg1, msg2, msg3)
```
### error
在控制台打印 error 日志
```ts
console.error(msg1, msg2, msg3)
```
### info
在控制台打印 info 日志
```ts
console.info(msg1, msg2, msg3)
```
### log
在控制台打印 log 日志
```ts
console.log(msg1, msg2, msg3)
```
### warn
在控制台打印 warn 日志
```ts
console.warn(msg1, msg2, msg3)
```
