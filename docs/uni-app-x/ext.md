# ext api

- [uni实人认证](https://uniapp.dcloud.net.cn/uniCloud/frv/dev.html)

- [打开文件](https://gitcode.net/dcloud/hello-uts/-/blob/dev/uni_modules/uts-nativepage/utssdk/app-android/index.uts)：uni.openDocument
