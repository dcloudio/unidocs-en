### 注意

uni-app x的定位，默认是系统定位，system。真机运行基座不包含三方定位sdk。

三方定位sdk方面，暂不支持高德、百度，但支持腾讯定位。可下载腾讯定位插件，在插件中配置key打包后生效。

如需其他定位，请在插件市场搜索定位相关的uts插件。

不管系统定位、还是三方sdk定位，都有很多注意事项，包括gms、坐标系、隐私和权限等，请仔细阅读下面的参考链接。
