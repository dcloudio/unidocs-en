## App平台差异  
App平台仅支持以像素值（px）设置字体大小，默认值为16px。仅对[text](uni-app-x/component/text.md)组件生效。  
> 不支持百分比的值  
> 不支持基于用户默认字体大小的绝对大小关键字，如small、medium、large等  
> 不支持em、rem、ex等单位
