## Tips

- 下载的文件存放在临时路径，此路径的文件会在下次App启动时删除，如有长期保存的需求，请指定filePath字段或者下载文件成功后转存至其他路径，转存文件暂时需要开发者开发UTS插件实现，后续我们会开放文件操作相关Api。
